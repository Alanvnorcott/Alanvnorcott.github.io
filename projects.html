NetScan is a Python project that I used to leverage my understanding of network configuration and security, as well as my proficiencies with Python. This script allows you to run a diagnostic of your network's IP, your machine's IP, or a custom IP of your choice. The scans included are: open port scanning, vulnerability scanning (via the CVE vulnerabilities database JSON file), network devices scan (ARP), ICMP sweeping, network traffic analysis (with packet capture file), and banner grabbing. I am hoping to create an appealing front end to this script as well but for the time being it must be utilized via the console. Most of these commands require administrator-level access, so if you decide to run this yourself keep that in mind.
